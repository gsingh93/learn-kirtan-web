Learn Kirtan
============

A web app for creating and sharing shabad notations.

Installation
------------

- Install [Node.js](https://nodejs.org)
- `npm install -g grunt-cli`
- `npm install`
- `grunt`

The resulting site will be in the `dist` folder.
